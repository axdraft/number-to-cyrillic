# Number To Cyrillic
Converts number to cyrillic

### Install
`npm install number-to-cyrillic`


### API

#### `convert(number)`
Converts an integer into a string.
```js
var converter = require('number-to-cyrillic');
converter.convert(21); // => “Двадцять одна гривня”
```
